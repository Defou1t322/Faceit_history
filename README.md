# FACEIT Match History

Tampermonkey-скрипт для анализа истории матчей FACEIT прямо в браузере — без сервера, без установки программ.

## Возможности

- **Мультипоиск** — несколько никнеймов сразу, общие матчи загружаются один раз
- **Определение пати** — ★ искомый, ◆ вероятная пати, + сокомандник (статистически)
- **Поиск по конкретным матчам** — вставь ссылки на матчи и найди пересечения игроков
- **Статистика** — частота встреч, средние K/D/A, роль каждого игрока
- **Фильтр периода** — последние N матчей / даты / смещение (избегает дублей при нескольких поисках)
- **Экспорт** — CSV, TSV (для Google Sheets), JSON

## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для Chrome / Firefox / Edge
2. Нажмите: **[Установить скрипт](https://raw.githubusercontent.com/Defou1t322/Faceit_history/main/faceit_history.user.js)**
3. В открывшейся странице нажмите **Install**
4. Перейдите на [faceit.com](https://www.faceit.com) — в правом нижнем углу появится кнопка **FH**

## Использование

1. Нажмите **FH** → введите API-ключ ([получить здесь](https://developers.faceit.com))
2. Введите никнеймы или ссылки на профили (по одному в строке)
3. Опционально: вставьте ссылки на конкретные матчи для анализа пересечений
4. Выберите игру и период → **Получить историю**

> API-ключ хранится только локально в Tampermonkey и отправляется исключительно на `open.faceit.com`.

## Обновление

Tampermonkey проверяет обновления автоматически. При новой версии скрипта появится уведомление — достаточно нажать «Обновить».

Принудительная проверка: Tampermonkey → Dashboard → скрипт → кнопка «Проверить обновления».

## Как искать подозрительных игроков

1. Поиск по нику жертвы (Y) → в **Статистике** смотрите колонку **против**: высокое значение при низком **вместе** = подозреваемый
2. Скопируйте ссылки на матчи, где подозреваемый X встречался с Y
3. Вставьте в поле **Конкретные матчи** → колонка **В заданных** покажет, кто повторяется во всех матчах
4. Одни и те же игроки рядом с X = вероятно организованная группа
